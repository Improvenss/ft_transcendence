
## Installation

You must be install; `docker.io`, `docker-compose`, `make`.

MacOS
```bash
$> brew install -y docker.io docker-compose make
```
Linux
```bash
$> sudo apt install -y docker.io docker-compose make
```
Windows
```bash
$> Use Linux, it's better.
```

## Running the app
```bash
$> make
```
or
```bash
$> docker-compose up --build
```
